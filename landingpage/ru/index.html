Plot Bunni

Инструмент для написания романов с открытым исходным кодом для организации идей, структурирования историй и использования ИИ для мозгового штурма и написания черновиков. Доступен прямо в вашем браузере, без проблем работает на настольных и мобильных устройствах!

Функции

Комплексный обзор и планирование романа

Визуализируйте структуру вашего романа с первого взгляда. Управляйте актами, главами и сценами с помощью интуитивно понятного интерфейса планирования. Связывайте элементы истории и поддерживайте порядок в сюжете.

  • Централизованная сетка управления романом.
  • Иерархическое планирование истории (акты, главы, сцены).
  • Подробная информация о романе: синопсис, автор, обложка.
  • Импорт планов историй из текста.

Богатое управление концепциями и сфокусированное письмо

Организуйте все строительные блоки вашей истории — персонажей, локации, лор и многое другое — в одном месте. Затем погрузитесь в свободное от отвлекающих факторов пространство для письма, созданное для концентрации, с полезными инструментами, такими как поддержка Markdown и удобная навигация.

  • Легко управляйте всеми элементами вашей истории, такими как персонажи, локации и лор.
  • Пишите свою историю сцена за сценой, используя гибкий редактор с поддержкой Markdown для форматирования.
  • Наслаждайтесь плавным процессом письма с текстовыми областями, которые растут вместе с вашим контентом, и встроенной помощью по форматированию.
  • Быстро переходите к любой части вашего романа с помощью интуитивно понятного навигатора по структуре.

Глубокая кастомизация и локальные данные

Настройте Plotbunni по своему вкусу с помощью обширных опций тем, включая светлый/темный режимы, предустановки и индивидуальную настройку цветов. Все ваши данные остаются в безопасности и доступны офлайн, хранясь локально в вашем браузере.

  • Светлый, темный и системный режимы темы.
  • Предопределенные пресеты тем (Сепия, Бумага и т.д.).
  • Полный контроль над отдельными цветами темы.
  • Локальное хранение данных через IndexedDB для офлайн-доступа.
  • Экспорт проектов романов (JSON) и контента (Markdown, Text, PDF).

Интеллектуальная помощь ИИ

Преодолейте писательский блок с помощью предложений ИИ для синопсисов сцен, планов, описаний и полного текста сцен. Настройте модели ИИ и промпты в соответствии с вашими потребностями.

  • Контекстные текстовые предложения ИИ.
  • ИИ-писатель романов для написания целых разделов.*
  • Настраиваемые профили конечных точек ИИ и промпты.
  • Потоковая передача предложений ИИ в реальном времени.

*ИИ-писатель романов является экспериментальной функцией и лучше всего подходит для быстрого прототипирования.
